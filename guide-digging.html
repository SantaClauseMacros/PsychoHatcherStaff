Advanced Digging Mode Troubleshooting

Psycho Hatchers Advanced Digging Mode Troubleshooting Guide

This guide is designed to help resolve issues with the Advanced Digging Mode macro. Follow these steps to identify and fix the problem.

Common Issues

Scaling is not set to 100%
Ask: "Is your display scaling set to 100%?"
Fix: Adjust display scaling to 100% in your PC's Display Settings (Settings > Display > Scale and Layout).
Roblox isn't in windowed mode
Ask: "Is Roblox in fullscreen mode?"
Fix: Switch Roblox to Windowed Mode by pressing Esc, navigating to Settings, and selecting Windowed in the Display Mode setting.
Not using the web browser version of Roblox
Ask: "Are you using the web browser version of Roblox?"
Fix: Close the Roblox app and switch to the web browser version.
Using a shiny hoverboard
Ask: "Are you using a shiny hoverboard?"
Fix: Switch to a basic hoverboard or dismount entirely. Shiny hoverboards can interfere with the macro.
Not in World 1
Ask: "Are you currently in World 1?"
Fix: Navigate to World 1. Advanced Digging Mode works only in this specific world.
Camera movement after starting
Ask: "Have you moved your camera since starting the macro?"
Fix: Rejoin World 1 and avoid moving the camera after starting the macro.

Uncommon Issues

Different camera mode than default
Ask: "Is your camera mode set to default?"
Fix: Go to Roblox settings and switch the camera mode to Default Classic.
PC settings that affect color
Ask: "Do you have any PC settings that alter color (Night Light, HDR, color filters)?"
Fix: Disable any color-altering settings on your PC.

Escalation Process

  • If common issues are fixed but the macro still doesn't work:
    • Ping Waktool for debugging issues related to PC or camera settings.
  • If the issue persists and cannot be resolved:
    • Ping another Support Member for further assistance.

Advanced Digging Mode

Psycho Hatchers Advanced Digging Mode Guide

This guide offers detailed steps to set up and efficiently use the Advanced Digging mode in Pet Simulator 99.

Setup Instructions

  1. Start in World 1: Begin your session in World 1 without moving the camera. This ensures the automation detects the game elements correctly.
  2. Unlock the Digsite: Confirm that the digsite is unlocked and accessible before running the automation.
  3. Disable Full-Screen Mode: Make sure Roblox is not running in full-screen mode. This prevents display issues, allowing the automation to function smoothly.
  4. Compatibility Across Resolutions: The automation is designed to work with all screen sizes and resolutions. No need to adjust your settings.
  5. Flexibility in Starting Locations: You can start from any world or digsite, making the automation convenient and adaptable.

Advanced Features

  • Optimized Click Speed: The automation ensures fast and accurate clicks for a seamless experience.
  • Smart Blueprints: The automation uses advanced blueprints to determine the most efficient path for digging.
  • No More Getting Stuck: The automation is designed to avoid common pitfalls, ensuring smooth operation throughout.

Using the Blueprint Grid

The blueprint grid is used by the automation for efficient digging:

  • Each square represents a tile in the digsite.
  • Black squares indicate priority spots for the automation to focus on for digging, as these positions maximize chest collection and avoid dead zones.

By following this grid pattern, the automation ensures a consistent and optimized digging strategy.

Automation Controls

  • Start: Press F1
  • Pause: Press F2
  • Exit: Press F3

Potential Issues

Lag
Ask: "Is your system experiencing lag?"
Fix: System lag can affect the automation's performance. Make sure your device is running smoothly to avoid this problem.
FPS Drops
Ask: "Are you experiencing significant FPS drops?"
Fix: Significant drops in frame rate can cause the automation to malfunction. Maintain a stable FPS for best results.
Best Shovel Requirement
Ask: "Are you using the best shovel available?"
Fix: For maximum efficiency, use the best shovel available (preferably the diamond shovel).

FAQ

Q: Why does the automation dig for a while and then go back home?
A: This is an intentional behavior designed to collect as many chests as possible efficiently.

Q: Why does it sometimes click on item pop-ups?
A: The automation uses color recognition, so it may occasionally click on item notifications. To prevent this, disable item pop-ups in your settings.
