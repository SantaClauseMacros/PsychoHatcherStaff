Anime Adventures Mode Guide: Farm Gems Efficiently

Psycho Hatchers Anime Adventures Mode Guide

This guide provides comprehensive setup and troubleshooting steps for the Anime Adventures Mode, which is designed for automating gameplay processes and farming gems efficiently in Anime Adventures.

Initial Setup: Font Configuration

Important First Step

Before using the Anime Adventures mode, change the Roblox font:

  1. Open Roblox
  2. Click the "Change Roblox Font" button in Psycho Hatcher
  3. Close Roblox completely
  4. Reopen Roblox for changes to take effect

Step 1: Adjust Unit Positions

  1. Prepare Your Position:
    • Join the game map without moving the camera (steady positioning is crucial for accuracy)
    • Press the "Respawn and move to starting position" button to reset your character's position
  2. Unit Configuration:
    • Add your units by clicking the "+" (Add new unit) button
    • Adjust their placement by dragging and moving the numbers shown on your screen

Step 2: Understanding the Unit Positions Table

The table consists of four key components:

  • X & Y Coordinates: Determines where units will be placed on the map
  • Slot: Specifies which unit to summon first
  • Wave: Determines when the unit should be summoned

Examples:

  • For Unit 3 placed on Wave 1: 1 | X & Y | Slot: 3 | Wave: 1
  • For Unit 3 summoned on Wave 4: 1 | X & Y | Slot: 3 | Wave: 4

Step 3: Managing Units Efficiently

  • "+" (Add new unit): Adds a unit at a specific location
  • "ðŸ—‘" (Delete the selected unit): Removes the selected unit from the list
  • "Slot <": Determines which unit from your available units will be placed in the selected position

Unit Properties:

  • Slot number: Select which unit you want for a specific location
  • Wave spawn: Choose which wave the unit should be summoned

Step 4: Auto-Exit Configuration

Set the option "Exit to lobby at wave number" to automate continuous grinding:

  • Locate the "Exit to lobby at wave number" option
  • Set it to the wave you prefer (for example, Wave 9)
  • The system will detect when you reach that wave, exit the match, and automatically rejoin for continuous grinding

Step 5: Save Your Settings

Important

After adjusting everything, remember to:

  • Click "Save unit positions" to keep the placements
  • Click "Save options" to store all other settings
  • Recommended: Restart the mode by pressing F3, then start it again for the settings to apply properly

Common Issues

Shiny Hoverboards Interference
Ask: "Are you using a shiny hoverboard?"
Fix: Avoid using shiny hoverboards as they can interfere with the macro functionalities. Use a basic hoverboard or dismount completely.
Not in World 1
Ask: "Are you in World 1?"
Fix: Navigate to World 1, which is essential for certain modes to function correctly.
Camera Movement
Ask: "Have you moved the camera after starting the macro?"
Fix: Rejoin World 1 and avoid moving the camera until the macro is fully operational.
Scaling and Window Mode
Ask: "Is your display scaling set to 100% and is Roblox in windowed mode?"
Fix: Set scaling to 100% in your PC's Display Settings and ensure Roblox is in windowed mode.
Browser Version
Ask: "Are you using the web browser version of Roblox?"
Fix: Use the web browser version of Roblox for optimal performance rather than the standalone app.
Default Camera Mode
Ask: "Is your camera mode set to Default Classic?"
Fix: Head to the Roblox settings and switch the camera mode to Default Classic.
PC Settings Affecting Color
Ask: "Do you have Night Light, HDR, or color filters enabled?"
Fix: Disable any PC settings that may be affecting the game's visuals, such as Night Light, HDR, or color filters.

Advanced Troubleshooting

Unit Configuration
Ask: "Have you properly configured all unit positions and wave settings?"
Fix: Double-check all unit configurations in the setup table, ensuring coordinates, slot numbers, and wave numbers are correctly entered.
Auto-Exit Configuration
Ask: "Have you set up the auto-exit feature correctly?"
Fix: Verify that the auto-exit wave number is properly configured to ensure continuous farming.
Mode Restart
Ask: "Did you restart the mode after making configuration changes?"
Fix: Press F3 to restart the mode after making any changes to the configuration.
Font Change
Ask: "Did you change the Roblox font and restart Roblox completely?"
Fix: Make sure you've clicked the "Change Roblox Font" button, closed Roblox completely, and reopened it.

Support Ticket Instructions

If you continue to face issues after trying all the above solutions:

  1. Record a video demonstrating the problem
  2. Include information about your system specifications
  3. Detail the troubleshooting steps you've already tried
  4. Submit this information through a support ticket for more tailored assistance
