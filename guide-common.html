Common Issues Troubleshooting

Psycho Hatchers Common Issues Troubleshooting Guide

This guide covers common issues that affect multiple modes in Psycho Hatcher. Use this as a reference for general troubleshooting.

Display and System Settings

Display Scaling
Issue: Many users forget to set their display scaling to 100%.
Fix: Ensure display scaling is set to 100% in Settings > Display > Scale and Layout. This setting is crucial for the macro to function properly.
Fullscreen vs. Windowed Mode
Issue: The game must be in windowed mode for proper operation.
Fix: Check if Roblox is in fullscreen mode and switch to windowed mode by pressing Esc, navigating to Settings, and selecting Windowed under Display Mode.
Web Browser Version Requirement
Issue: Users often use the app version instead of the required web browser version.
Fix: The web browser version of Roblox is required for optimal performance. If using the app version, close it and utilize the browser version instead.
PC Language Settings
Issue: Non-English language settings can cause recognition problems.
Fix: PC language must be set to English (US). Change this in Settings under Time & Language if needed.

Game-Specific Issues

Macro UI Overlap
Issue: Macro interface overlapping with the Roblox game interface.
Fix: Adjusting the window size or repositioning the macro can resolve this problem. Ensure both interfaces are clearly visible.
Hoverboard Issues
Issue: Shiny hoverboards can interfere with macro functionality.
Fix: Switch to a basic hoverboard or dismount entirely when using macros that are affected by hoverboard visibility.
World Restrictions
Issue: Using a mode in the wrong world location.
Fix: Some modes, like the Treehouse Mode, only operate in specific worlds (e.g., World 1). Ensure you are in the correct world for the selected mode.
Camera Movement
Issue: Moving the camera after joining can disrupt macro functionality.
Fix: Avoid moving your camera after joining the game. If you accidentally moved it, rejoin without moving your camera.

Performance Issues

System Lag
Issue: System performance affecting macro timing and recognition.
Fix: Close unnecessary background applications. Reduce Roblox graphics settings to improve performance.
Browser Performance
Issue: Browser resource usage affecting game performance.
Fix: Close unused browser tabs and extensions. Consider restarting the browser if performance issues persist.

General Troubleshooting Process

  1. Verify all system settings (scaling, language, etc.)
  2. Check game-specific settings (windowed mode, graphics)
  3. Confirm you're in the correct location for the selected mode
  4. Ensure all requirements for the specific mode are met
  5. If problems persist, contact any available staff member
