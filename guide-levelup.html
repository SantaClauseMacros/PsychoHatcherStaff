Level Up Mode Troubleshooting

Psycho Hatchers Alt Level Up Mode Troubleshooting Guide

This guide will help you identify and resolve common issues with the Alt Level Up Mode.

Common Issues

Scaling is not set to 100%
Ask: "Is your display scaling set to 100%?"
Fix: Ensure that your display scaling is set to 100%. You can check this in your PC settings under Display Settings.
Not using web browser version
Ask: "Are you using the web browser version of Roblox?"
Fix: Confirm that you are running the web browser version of Roblox; if not, switch your game to this version.
Roblox is in fullscreen mode
Ask: "Is Roblox running in fullscreen mode?"
Fix: If Roblox is running in fullscreen, press Esc, go to Settings, and change the Display Mode to "Windowed."
Camera movement after joining PS99
Ask: "Did you move your camera after joining the PS99 game?"
Fix: Avoid moving your camera after joining the PS99 game. If you accidentally did this, the solution is to rejoin PS99 without moving your camera.
F1 Key not pressed
Ask: "Did you press F1 after loading into the game?"
Fix: After loading into the game, ensure you have pressed F1, which initializes the macro setup.
PC language not set to English (US)
Ask: "Is your PC language set to English (US)?"
Fix: It's important to have your PC language set to English (US). Change this in the Settings under Time & Language.

Uncommon Issues

System performance issues
Ask: "Is your system experiencing lag or performance issues?"
Fix: Close unnecessary applications and optimize system performance for smoother macro operation.
Incorrect game version
Ask: "Are you using the latest version of the game?"
Fix: Ensure you are using the most recent game version, as outdated versions may cause compatibility issues.

Escalation Process

If all troubleshooting steps have been completed and the issue persists:

  • Contact any available staff member for assistance with technical issues.
  • For advanced troubleshooting, please escalate to a senior support team member.
