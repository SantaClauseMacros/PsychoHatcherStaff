Open Stuff Mode Guide

Psycho Hatchers Open Stuff Mode Guide

This guide provides setup instructions for the Open Stuff Mode, which is designed to help users open items at an accelerated speed, reaching up to 10,000 times the normal rate.

Setup Instructions

  1. Enable the Controller Icon: Locate the controller icon on the right side of the Psycho Hatcher interface. You must turn on this icon before running the automation.
  2. Navigate the User Interface: Use the arrow keys to select the item you wish to open.
  3. Choose Opening Speed: Select the desired speed for opening items: "10," "100," "1,000," or "10,000" times the normal speed.
  4. Run the Mode: Once you've configured your settings, activate the mode to begin opening items at the accelerated rate.

Troubleshooting

Controller Not Functioning
Ask: "Is the controller icon visible and active in the interface?"
Fix: Ensure the controller icon is enabled. If the controller is still not responding, you may need to download and install the necessary driver from GitHub for proper controller support.
Items Not Opening at Selected Speed
Ask: "Have you properly selected the speed multiplier using the arrow keys?"
Fix: Verify that you've correctly selected the desired speed option ("10," "100," "1,000," or "10,000"). Try selecting a different speed and then returning to your preferred choice.
Interface Navigation Issues
Ask: "Are you having trouble selecting items with the arrow keys?"
Fix: Check if any other keyboard controls or macros are active that might be interfering with the arrow key inputs. Try disabling other modes temporarily.

Best Practices

  • Start with lower speed multipliers to test functionality before using the maximum 10,000x speed.
  • Make sure your system resources aren't overly taxed when using higher speeds to prevent crashes.
  • Keep track of your inventory space to prevent potential issues with item overflow.

Important Note

Using extremely high speed multipliers (10,000x) may cause temporary lag in some systems. If you experience performance issues, try reducing the speed multiplier.
