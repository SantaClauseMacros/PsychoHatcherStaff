Treehouse Mode Guide

Psycho Hatchers Treehouse Mode Guide

This guide provides instructions on how to set up and use the Treehouse mode effectively in Pet Simulator 99.

Setup Instructions

  1. Start in Any World: You can begin in any world, but if you are starting in World 1, do not move your camera, as it will disrupt the automation's functionality.
  2. Avoid Using a Shiny Hoverboard: Ensure you use a regular hoverboard to avoid issues.

What Not to Do

To ensure smooth operation, avoid the following:

  • Do Not Use Any Speed Potions: Speed potions can interfere with the automation.
  • Do Not Run Without Secret Keys: Ensure you have secret keys available, as they are necessary for the automation's operation.
  • Make Sure You Are Rebirth 3: You must be at Rebirth 3 to access and open the treehouse.

Common Issues

Scaling is not set to 100%
Ask: "Is your display scaling set to 100%?"
Fix: Adjust scaling to 100% in your PC's Display Settings (Settings > Display > Scale and Layout).
Roblox isn't in windowed mode
Ask: "Is Roblox in fullscreen mode?"
Fix: Switch to Windowed Mode in Roblox by pressing Esc, opening Settings, and selecting Windowed under Display Mode.
UI of the macro overlaps Roblox
Ask: "Is the macro interface overlapping your Roblox screen?"
Fix: Adjust the macro's window size or reposition it to avoid overlap with the Roblox game interface.
Using a shiny hoverboard
Ask: "Are you using a shiny hoverboard?"
Fix: Switch to a basic hoverboard or dismount entirely. Shiny hoverboards can interfere with macro functionality.
Not in World 1
Ask: "Are you in World 1?"
Fix: Navigate to World 1, as Treehouse Mode works best in this world.
Not using the web browser version of Roblox
Ask: "Are you using the web browser version of Roblox?"
Fix: Close the Roblox app and use the browser version instead.
Lag issues
Ask: "Are you experiencing lag during gameplay?"
Fix: Lag can cause significant problems with any automation. Ensure your system is running smoothly and not experiencing lag.

Uncommon Issues

OCR Fails
Description: The OCR (Optical Character Recognition) function of the macro might fail to detect on-screen elements correctly.
Fix: Ping another Support Member or Waktool to debug the OCR setup.
Big Games bugs causing zones after Zone 1 not to load
Description: Sometimes, Big Games servers may bug out, failing to show zones beyond Zone 1.
Fix: If this happens, ping another Staff Member or Waktool for assistance.
